# QNRTCLinux Release Notes for 5.0.0

## 简介
QNRTCLinux 是七牛云推出的一款适用于 Linux 平台的实时音视频 SDK，提供了灵活的接口，支持高度定制以及二次开发。

## 版本
- 发布 libqn_rtc.so

## 功能
- 支持基本的音视频通话 
- 支持导入 PCM/YUV 数据
- 支持纯音频通话 
- 支持网络自动重连
- 支持音视频数据的回调
- 支持媒体传输协议的配置

## 注意事项
- 线上仅提供 Ubuntu Desktop x86_64 版本
- [Demo 源码 GitHub 地址](https://github.com/pili-engineering/QNRTC-Linux)
